第２回：ブロックを設置しよう

初めてのプログラミングは、ブロックを設置するところから始まります。
この作業にはプログラミングの基礎がぎっしりと詰まっていますから、じっくり理解しながら取り組みましょう。
今回やることは次の通りです。

  • ブロックの設置の仕方を学ぼう
  • 変数を使ってブロックを設置してみよう
  • 配列の仕組みを理解しよう
  • 配列を使ってブロックを複数設置してみよう
  • 二次元配列でブロックを設置する

ブロックの設置の仕方を学ぼう

まずブロックの設置の仕方をご紹介します。
この方法は、p5.jsを使用しないとできませんのでご注意ください。
では以下と同じ様にプログラムを書いてください。

sketch.js
function setup(){
    createCanvas(600,400);
}

function draw(){
    background(0);
    fill(255);
    rect(0,0,100,100);
}


書けましたでしょうか。それでは第１回と同様に、「ブロック崩し」フォルダから「index.html」をダブルクリックしてみましょう。
下の画像の様な画面は表示されましたか？



もしそうでなかった場合には、打ち間違えが発生しているかもしれません。上のプログラムをコピペして、ブラウザの再読み込みをしてみましょう。
ブロックを描画するにあたり、描画するための「カンバス」や「画用紙」の様なものが必要です。
それを "createCanvas()" で作り出しています。また、()内の数字はその大きさを表しています。
,(カンマ)の左側が横幅、右側が縦幅を表します。つまり、横幅が600で縦幅が400ということです。
