第２回：ブロックを設置しよう


初めてのプログラミングは、ブロックを設置するところから始まります。
この作業にはプログラミングの基礎がぎっしりと詰まっていますから、じっくり理解しながら取り組みましょう。
今回やることは次の通りです。

  • ブロックの設置の仕方を学ぼう
  • 変数を使ってブロックを設置してみよう
  • 配列の仕組みを理解しよう
  • 配列を使ってブロックを複数設置してみよう
  • 二次元配列でブロックを設置する

ブロックの設置の仕方を学ぼう


まずブロックの設置の仕方をご紹介します。
この方法は、p5.jsを使用しないとできませんのでご注意ください。
では以下と同じ様にプログラムを書いてください。

sketch.js
function setup(){
    createCanvas(600,400);
}

function draw(){
    background(0);
    fill(255);
    rect(0,0,100,100);
}


書けましたでしょうか。それでは第１回と同様に、「ブロック崩し」フォルダから「index.html」をダブルクリックしてみましょう。
下の画像の様な画面は表示されましたか？



もしそうでなかった場合には、打ち間違えが発生しているかもしれません。上のプログラムをコピペして、ブラウザの再読み込みをしてみましょう。

では解説いたします。
ブロックを描画するにあたり、描画するための「カンバス」や「画用紙」の様なものが必要です。
それを "createCanvas( )" で作り出しています。また、( )内の数字はその大きさを表しています。
カンマ（ , ）の左側が横幅、右側が縦幅を表します。つまり、横幅が600で縦幅が400ということです。
また、( )内に記述される値のことを引数といいます。
600の様に一番目に記述される引数のことを第一引数、400の様に二番目に記述される引数のことを第二引数と言います。
これらは関数を勉強するときに詳しく取り上げます。

次にrect( )。　これは四角形を描くものです。引数は左から順番に X座標, Y座標, 横幅, 縦幅 です。

最後に、P5.jsを使うにあたり最も大切な「function setup( ){ }」と「function draw( ){ }」を解説します。
まず、function setup( ){ } の{ }内(中カッコ内)に記述されているプログラムが最初に一回だけ実行されます。
次に、function draw( ){ }の{ }内に記述されているプログラムが常に実行されます。

これらを頭の方隅において、次の章に進みましょう。

変数を使ってブロックを設置してみよう


ここからは、変数を使ってブロックを設置してみましょう。
そもそも変数とは何でしょうか。これは、値を保存しておく箱のようなもので、
その箱に名前をつけ値を代入することで、その名前が値の代わりになります。

sketch.js
let school = "神大附属";

console.log(school);

//console.log("神代附属"); と結果が一緒

上のプログラムを function draw( ){ }の外に書いてみましょう。
これは変数schoolに"神大附属"という文字列を代入しています。
これで変数schoolは "神大附属" として使える様になったわけです。
console.log()ですが、これはブラウザのコンソールにカッコ内の値を表示させるものです。したがって、コンソールには 神大附属と表示されるはずです。
ブラウザを開いて右クリック。表示されたオプションの中に「開発者モード」「デベロッパーツール」「Inspect」のいずれかがあったら、それをクリックしてください。
下の画像の様になりましたか？ブラウザの右側にこの画面が出ていれば、コンソールの表示に成功しています。
さらに神大附属が出力されていればオッケーです。



さて、本題に移ります。
先ほど説明した様に、変数は値を保存しておく箱の様なものです。
では、なぜこれが便利なのかを実際に見ていきましょう。

例えば、2つのブロックを縦に並べて配置したいとします。そうすると、二のブロックのX座標は一致します。
この時に、同じ変数を使ってあげると便利なのです。下のプログラムを書いて実行してみてください。
ブロックが縦二つに並んでいると思います。
sketch.js
let x=0;

function setup(){
    createCanvas(600,400);
}

function draw(){
    background(0);
    fill(255);
    rect(x,0,100,100);
    rect(x,100,100,100);
}

試しに let x=0; を let x=100; に変えてみましょう。
変わらずブロックが縦に並んでいるはずです。この便利な変数を縦横にブロックを配置するのに応用していきます。
